Row Reduction puzzles

{% endblock %} {% block method %}

The Row Reduction method is similar to the checkRows method. To use the Row Reduction method, pick a row, and then pick a number that is missing from that row. Then, test that number in all unfilled cells in that row to see if it is a possibility. If all of the possible cells for the missing number are in the same box, cells in that box that are not in the row cannot contain that particular number, and that candidate can be eliminated as a possibility.
